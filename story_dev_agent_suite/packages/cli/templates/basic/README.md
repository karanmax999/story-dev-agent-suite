# Story Protocol App

A minimal Story Protocol application using the `story-client` package.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Configure environment:
```bash
cp .env.example .env
# Edit .env and add your STORY_PRIVATE_KEY
```

3. Run the example:
```bash
npm run dev
```

## Usage

The example in `src/index.ts` demonstrates how to:
- Initialize the Story Client
- Register an IP Asset
- Handle errors
